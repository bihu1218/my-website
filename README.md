# AI 家居风水顾问移动端

移动优先的 AI 家居风水顾问 Web App。用户可以在手机里拍照或上传家中照片，补充房屋、家庭和命理参考信息后生成一份结构化报告。

## 产品原则

- 用建筑动线、采光通风、环境心理、人体工学解释传统风水术语。
- 不做恐吓式判断，不承诺改命暴富，不强推摆件。
- 建议优先低成本、可执行、可验证。

## 本地运行

```bash
npm install
npm run dev
```

手机同一局域网访问：

```bash
npm run dev:host
```

终端会显示局域网地址，例如 `http://192.168.x.x:5173/`。

## 接入真实图像分析

没有密钥时，应用会使用本地规则引擎生成演示报告。

要启用 AI 图像分析，在启动前设置：

```bash
$env:OPENAI_API_KEY="你的密钥"
npm run dev:host
```

可选覆盖模型：

```bash
$env:OPENAI_MODEL="gpt-5.5"
```

后端接口在 `POST /api/analyze`，密钥只在本地服务端使用，不会暴露到浏览器。

## 当前功能

- 手机拍照/相册上传，最多 6 张。
- 上传前自动压缩图片，减少手机照片过大导致的请求失败。
- 选择客厅、卧室、厨房、书房、玄关或整屋。
- 补充朝向、面积、大门朝向、楼层、入住年份、居住类型、家庭情况、主卧使用者。
- 填写出生日期、出生时间、出生地、性别作为命理匹配参考。
- 选择求财、健康、夫妻关系、事业、孩子学习、睡眠、整体舒适度等诉求。
- 生成移动端报告：整体评分、客厅建议、卧室建议、厨房建议、财位/事业/健康建议、个性化颜色材质摆件建议。
- 报告会保存到本机浏览器历史，最多 12 份。
- 支持下载 Markdown 报告。

## 第一版顾问逻辑

这不是训练一个“风水模型”，而是把大模型包装成专业顾问系统：

1. AI 先根据图片识别门、窗、床、沙发、灶台、镜子、采光、通风、杂乱程度和动线。
2. 再叠加规则知识库：形峦风水、八宅参考、玄空飞星参考、五行命理参考、家居美学和现实安全规则。
3. 输出可执行建议：问题、风水解释、现代环境解释、调整方法、优先级、花费、难度和预期效果。
4. 命理信息只作为参考；没有出生时间和出生地时，八字/五行判断会明显变弱。
