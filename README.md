# AI获客体检器

这是一个静态 H5 MVP，用一个入口承接两类人群：

- 中小老板 / 创业者：诊断业务获客链路。
- 新媒体获客 / 知识付费：诊断内容转化链路。

## 使用

直接用浏览器打开：

```text
index.html
```

## 当前版本

- 人群分流
- 8 题诊断
- 自动评分
- 诊断类型
- 7 天行动清单
- 短视频选题
- 私信承接话术
- AI 化优先动作
- 留资表单
- 浏览器本地保存线索
- CSV 导出
- 微信/企微二维码上传
- 报告海报 PNG 下载
- 私信话术复制
- 大模型配置面板
- 自动获取模型列表并按场景匹配模型
- AI 深度方案生成

## 本地线索说明

当前版本不需要后端。保存的线索会进入当前浏览器的 `localStorage`，适合测试阶段使用。

注意：

- 换浏览器或清理缓存后，本地保存的线索会丢失。
- 正式投放前，建议接入飞书表格、腾讯文档、企业微信或自己的后端接口。
- 二维码图片也保存在当前浏览器本地，只用于当前设备预览和生成海报。

## 大模型配置说明

报告页有一个“AI 深度方案（可选）”模块，支持小鸡聚合AI或其他 OpenAI 兼容接口。

使用步骤：

1. 打开小鸡聚合AI控制台，复制 Base URL、API Key 或模型名。
2. 回到报告页，填写 Base URL 和 API Key。
3. 选择自动匹配场景：深度诊断、低成本快速、文案生成。
4. 点击“获取模型并自动匹配”。
5. 点击“生成AI深度方案”。

注意：

- API Key 只保存在当前浏览器本地。
- 如果浏览器提示跨域或网络错误，说明模型平台不允许前端直连，需要后续增加一个后端代理。
- 没配置模型时，本地规则诊断仍然可以完整使用。

## 测试流程

1. 选择人群。
2. 完成 8 道题。
3. 生成报告。
4. 上传微信或企微二维码。
5. 填写称呼、微信/手机号、行业方向并保存线索。
6. 点击导出 CSV，检查线索是否完整。
7. 点击下载海报，检查二维码和诊断结果是否显示正常。
8. 如需深度方案，配置模型并生成 AI 方案。

## 后续建议

1. 接入后端或飞书/腾讯文档，跨设备保存测试结果。
2. 增加后端代理，保护 API Key，并解决浏览器跨域限制。
